## [0.1.0] - 2026-06-06

### Agregado
- Estructura inicial del proyecto.
- Generación de datos sintéticos de viviendas en Lima.
- Módulo de preprocesamiento.
- Modelo RandomForestRegressor.
- Tests automáticos con pytest.
- Workflow de GitHub Actions con Python 3.10 y 3.11.
- Entrenamiento automático y generación de métricas.
- Artefacto descargable `metricas-modelo`.

### Notas
- El proyecto usa datos sintéticos.
- Las métricas pueden variar ligeramente entre ejecuciones.
